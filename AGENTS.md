# Règles du projet Expow

- **Tests unitaires :** Toujours exécuter le script `./tests/run_tests.sh` et vérifier qu'il passe avec succès *avant* de réaliser le moindre `git commit`.
- **TDD / Qualité :** À chaque fois que vous corrigez un bug (fix) ou ajoutez une fonctionnalité, vous DEVEZ ajouter un test correspondant dans `./tests/run_tests.sh` pour prouver que ça fonctionne et éviter les régressions.
